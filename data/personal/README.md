# Personal layer / Capa personal

Your personal criterion lives here. It is **gitignored by design** — private,
local-first, never published by accident.

To deliberately share a case with the community, use:

```
criteria promote <case-id>
```

**ES:** Tu criterio personal vive aquí. Está ignorado por git a propósito:
privado y local. Publicar un caso es siempre un acto explícito con
`criteria promote`.
